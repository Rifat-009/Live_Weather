# 📖 Overview

**Live Weather** is a responsive web application that allows users to search for any city and view its current weather conditions in real time. The application retrieves weather information from an external Weather API and displays it in a clean, interactive, and user-friendly interface.

---

# ✨ Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Weather condition and icons
- 💧 Humidity information
- 🌬️ Wind speed
- 🌅 Sunrise & Sunset
- 📱 Responsive design
- ⚡ Fast API integration
- ❌ Error handling for invalid locations

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Rifat-009/Live_Weather.git
```

Navigate to the project folder

```bash
cd Live_Weather
```

Open `index.html` in your preferred browser.

# 🔑 API Configuration

If your project uses an API key:

1. Create an account on your preferred Weather API provider.
2. Generate your API key.
3. Replace the placeholder key inside your JavaScript file.

Example:

```javascript
const API_KEY = "YOUR_API_KEY";
```
